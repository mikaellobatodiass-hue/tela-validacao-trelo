# 🔐 Tela de Login - Clone Trello

Este projeto consiste na criação de uma interface de autenticação inspirada na tela de login do Trello, desenvolvida do zero com foco em fidelidade visual, organização de código e boas práticas de front-end.

## 🚀 Sobre o Projeto

A proposta deste projeto foi recriar a experiência visual da tela de login do Trello, utilizando apenas HTML e CSS, sem o uso de frameworks.

O objetivo principal foi:
- Praticar estruturação semântica
- Trabalhar layout responsivo
- Aprimorar habilidades com CSS moderno (Flexbox)
- Desenvolver autonomia na construção de interfaces reais

---

## 🎯 Funcionalidades

- Formulário de login com campos de email e senha
- Botões de autenticação social (Google, Microsoft, Apple, Slack)
- Layout inspirado no Trello
- Estrutura limpa e organizada
- Design responsivo básico

---

## 🛠️ Tecnologias utilizadas

- HTML5
- CSS3
- Flexbox

## 💡 Decisões de Desenvolvimento

Este projeto foi construído sem o uso de bibliotecas externas para reforçar o domínio dos fundamentos.

Alguns pontos importantes:
- Uso de Flexbox para alinhamento e distribuição de elementos
- Separação clara entre estrutura (HTML) e estilo (CSS)
- Código organizado visando manutenção e escalabilidade
